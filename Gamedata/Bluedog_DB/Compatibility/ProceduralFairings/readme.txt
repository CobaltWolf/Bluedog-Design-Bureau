BDB-ProceduralFairings
Procedural fairings for CobaltWolf's Bluedog Design Bureau.
By JSO

Converts all BDB fairing bases to work with procedural fairings.
Included side textures:
Bluedog default
Bluedog Titan
Etoh
Etoh Test

Dependencies
Procedural Fairings
